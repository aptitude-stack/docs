# Org Profile Metadata

This file captures the recommended GitHub org profile metadata that cannot be set from repository files alone.

## Recommended Pinned Repositories

Pin these first:

1. `.github`
2. `aptitude-server`
3. `aptitude-client`
4. `aptitude-docs`

When `aptitude-publisher` exists as a separate public repository, replace `aptitude-docs` or expand the pinned set if you have room.

## Recommended Repository Descriptions

### `.github`

Org profile, shared docs, architecture references, and community health files for Aptitude.

### `aptitude-server`

Authoritative registry backend and public HTTP API for publishing, discovery, fetch, governance, and audit.

### `aptitude-client`

Python MCP and CLI runtime client for discovery, solving, lock generation, and execution planning.

### `aptitude-docs`

Product, architecture, and design documentation for the Aptitude ecosystem.
