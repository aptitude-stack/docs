## Learned User Preferences

- When updating the root `README.md`, keep a product-wide Aptitude overview with slightly technical language (explicit publisher, registry, and resolver ownership, flows, and contracts-oriented framing).
- In the README documentation map, use absolute GitHub file URLs under `https://github.com/aptitude-stack/docs/blob/main/...` when linking to files in this repository.

## Learned Workspace Facts

- Git remote `origin` is `https://github.com/aptitude-stack/docs.git`.
- The canonical root product overview file is `project-overview.md` (hyphenated filename at repo root).
- `.gitignore` is set to ignore all paths except directories, `*.md` files, and `.gitignore` itself, so non-Markdown files are not intended to be tracked by default.
